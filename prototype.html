TiPEX: a tool chain for Timed Property Enforcement during eXecution

TiPEX Tool Overview

TiPEX is a tool of 1200 LLOC in Python and consists of three modules (shown in the figure below). The tool also uses the PyUPPAAL, DBMpyuppaal, and PyDBM libraries which are available for download at http://people.cs.aau.dk/~adavid/python/.

The Enforcement Monitor from Timed Automata (EMTA) module consists in an implementation of the enforcement algorithms.

The Enforcement Monitor Evaluation (EME) module is a test harness for the performance of enforcement monitors, with functionalities such as a trace generator.

The Timed Automata Generator (TAG) module provides functionalities such as generating and composing TAs (which can be also used in other contexts than enforcement monitoring).

The source code and some examples can be downloaded here.
